# TeluguBudget 🚀

TeluguBudget is a budget-focused affiliate shopping website that helps users discover trending products, daily deals, and useful items at the best prices.

## Features

- 🔥 Today's Deals
- 💰 Products Under ₹150
- ❤️ Impress Your Crush
- 👩 Mom's Favorites
- 👨 Dad's Essentials
- 🕉️ Devotional Products
- 🛒 Amazon & Flipkart Affiliate Links
- 📱 Mobile-friendly design
- 🔎 Product search
- 🖼️ Multiple product images
- ⭐ Ratings and product details
- 🛠️ Admin product management

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- Vercel
